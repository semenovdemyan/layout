Требования к тестовому заданию:

1. Основная сложность этого задания заключена в том, 
чтобы самостоятельно разобраться с нетипичным для веб-разработки инструментом для чтения AI-файлов(Adobe Illustrator).

2. PP - не важен (строго следовать размерностям в макете не следует), 
важно самостоятельно выбрать и сопоставить размеры для того, что бы результат был максимально приближен к макету.

3. Минимум медиа-запросов, желательно только 1. 
Так же при любом разрешении экрана тексты не должны выезжать за рамки, все должно смотреться профессионально.

4. Выбор инструментов и фреймворков для реализации не важен. 
Можете использовать даже чистые HTML,CSS(JS не обязательно). 
Главное, что бы финальный результат был прост и удобен для просмотра.

5. Шрифт - Gilroy, но подойдет любой похожий, рубленый, без засечек.
